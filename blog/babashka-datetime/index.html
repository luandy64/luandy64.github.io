Babashka Datetime

Last updated : 2023-10-15T14:39:22-04:00

If I need a datetime in Python, I reach for

from datetime import datetime

datetime.now()

So I need something as quick for clojure.

Apparently, it’s just

(-> (java.time.LocalDate/now)
     str)
;; => "2023-10-15"

#babashka
